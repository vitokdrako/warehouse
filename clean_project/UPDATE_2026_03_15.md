## Backend Update
**Папка:** `clean_project/backend_update/`
**URL:** `backrentalhub.farforrent.com.ua`

### Змінені файли:
- `routes/documents.py` — додано Акт взаєморозрахунків
- `routes/finance.py` — додано закриття місяця, нарахування шкоди/прострочки
- `routes/admin.py` — додано settlement_act в шаблони
- `services/company_config.py` — без змін
- `templates/documents/settlement_act.html` — НОВИЙ шаблон

### Видалені файли (застарілі):
- `routes/damages.py` — ВИДАЛЕНО
- `routes/audit.py` — ВИДАЛЕНО
- `server.py` — оновлено (прибрано імпорти damages/audit)

### Нова таблиця БД:
`monthly_reports` — створюється автоматично при першому використанні

### Нові API ендпоінти:
- `GET /api/documents/settlement-act/{order_id}/preview` — Акт взаєморозрахунків
- `GET /api/documents/settlement-act/{order_id}/pdf` — PDF версія
- `POST /api/finance/close-month` — Закрити місяць
- `GET /api/finance/monthly-reports` — Список звітів
- `GET /api/finance/monthly-reports/{id}` — Деталі звіту
- `DELETE /api/finance/monthly-reports/{id}` — Видалити звіт

## Порядок деплою:
1. Завантажити `backend_update/` на сервер
2. Видалити `routes/damages.py` та `routes/audit.py` з сервера
3. Замінити `server.py` (прибрано імпорти)
4. Замінити `routes/documents.py`, `routes/finance.py`, `routes/admin.py`
5. Додати `templates/documents/settlement_act.html`
6. Перезапустити бекенд
7. Замінити фронтенд білд
