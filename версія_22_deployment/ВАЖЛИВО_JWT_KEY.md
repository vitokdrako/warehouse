# ВАЖЛИВО! Налаштування JWT ключа

## Проблема
Токени створюються з одним секретним ключем, а перевіряються з іншим.

## Рішення

### На продакшн бекенді додайте в `.env`:

```
JWT_SECRET_KEY=ваш-секретний-ключ-який-ви-обрали
```

### Або згенеруйте новий:

```bash
python3 -c "import secrets; print(secrets.token_hex(32))"
```

### Після зміни:
1. Перезапустіть бекенд
2. **Вийдіть з системи (logout)**
3. **Увійдіть заново (login)**

Це створить новий токен з правильним ключем.

---

## Альтернатива (швидке виправлення)

Просто вийдіть і увійдіть заново на продакшені!
Новий логін створить токен який буде працювати.
